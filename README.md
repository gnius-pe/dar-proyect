## Instalación

1.- Clona este repositorio en tu máquina local:
   ```bash
   git https://github.com/gnius-pe/dar-proyect.git
   ```

2.- Instalaciones 
   ```bash
   npm install
   ```

3.- Ejecucion del servidor
   ```bash
    node app.js
   ```
